# Ansible

A playbook used to setup a new machine.

## How to

1. Install Homebrew
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

2. Install Git:
```bash
brew install git
```

3. Install Ansible:

```bash
brew install ansible
```

4. Clone the repo:
```bash
git clone git@github.com:SombreShinobi/ansible.git
```
